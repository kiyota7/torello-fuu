# タスクボード

Trello風のカンバン方式タスク管理Webアプリケーション。個人でカフェ開業に向けた準備を進める
依頼者からの相談を想定した学校課題として開発している(詳細は[要件定義書](要件定義書.md)参照)。

## 技術スタック(概要)

- フロントエンド: React(Vite, JavaScript)

## セットアップ・起動方法

前提条件や詳細な手順は[ローカル開発環境セットアップ](docs/ローカル開発環境セットアップ.md)を参照。

```bash
# PostgreSQL起動
cd backend
docker compose up -d

# バックエンド起動(別ターミナル)
cd backend
mvn spring-boot:run

# フロントエンド起動(別ターミナル)
cd frontend
npm install
npm run dev
```

起動後、`http://localhost:5173` でボード画面(READ表示)を確認できる。
`http://localhost:8080/api/health` でバックエンドの疎通確認ができる。

## 実装状況

- [x] カード内容の編集・ステータス(所属リスト)変更(`PUT /api/cards/{cardId}`)。カードをクリックすると
      詳細モーダルが開き、内容編集とセレクトボックスによるリスト変更ができる
- [ ] ログイン・新規登録・ログアウト(F-13〜F-15)
- [ ] カードの追加・削除(F-04, F-06)
- [ ] リストの追加・編集・削除(F-01〜F-03)
- [ ] 優先度順・期限順への並び替え、自由なドラッグ&ドロップ(F-07, F-12)

ログインAPI未実装のため、現状はテストユーザー固定でボードを表示している。
