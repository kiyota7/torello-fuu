# 非機能要件定義書

タスク管理アプリ「タスクボード」の非機能要件。

| 項目 | 内容 |
|---|---|
| 動作環境 | 最新版のGoogle Chrome、Microsoft Edge等のモダンブラウザ |
| 実行方法 | バックエンド(Spring Boot)・データベース(PostgreSQL)を起動した上で、フロントエンド(React)にブラウザからアクセスして利用する |
| レスポンス性能 | 操作(追加・編集・削除・移動)に対して体感的な遅延なく画面へ反映されること |
| データ保存範囲 | データはバックエンド経由でPostgreSQLデータベースに保存され、ユーザーごとに分離される(他ユーザーのデータは参照できない) |
| ユーザビリティ | 直感的なクリック操作・ドラッグ操作のみで、マニュアルなしに利用できること |
| セキュリティ | パスワードはハッシュ化(bcrypt等)して保存する。認証はセッション(Cookie)を使用する。個人情報として扱うのはユーザー名・パスワードのみとし、その他の機密情報は扱わない |
