# 画面設計書

タスク管理アプリ「タスクボード」の画面設計。
機能の詳細は [機能一覧.md](機能一覧.md) を参照。

## 画面一覧

本システムは以下の3画面で構成される。

| 画面ID | 画面名 | 概要 |
|---|---|---|
| S-01 | ログイン画面 | ユーザー名・パスワードを入力してログインする画面(未ログイン時の初期画面) |
| S-02 | 新規登録画面 | ユーザー名・パスワードを入力して新規アカウントを作成する画面 |
| S-03 | ボード画面 | リストとカードを表示・操作するメイン画面(ログイン後に表示) |

## 画面構成要素

### S-01 ログイン画面 / S-02 新規登録画面

- タイトル、ユーザー名入力欄、パスワード入力欄、送信ボタン(「ログイン」/「登録する」)
- エラーメッセージ表示欄(ユーザー名・パスワード誤り、ユーザー名重複時)
- もう一方の画面への切り替えリンク(「アカウントをお持ちでない方はこちら」等)

### S-03 ボード画面

- ヘッダー: アプリタイトル、ログイン中のユーザー名表示、「サンプルに戻す」ボタン、「ログアウト」ボタン
- リスト(複数): リスト名、「優先度順」「期限順」並び替えボタン、カード一覧、「+ カードを追加」ボタン、リスト削除ボタン
- カード(各リストに複数): カード本文、優先度(高・中・低)、期限日、カード削除ボタン
- 「+ リストを追加」: 画面右端に常に表示される、新規リスト作成用の入力欄

## 画面遷移

これまで単一画面構成だったが、ログイン機能の追加により以下の画面遷移が発生する。

- 未ログイン時はS-01(ログイン画面)を表示する
- S-01とS-02は切り替えリンクで相互に遷移する
- S-01でログインに成功、またはS-02で新規登録に成功すると、自動的にS-03(ボード画面)に遷移する
- S-03で「ログアウト」を押すと、S-01(ログイン画面)に戻る
- S-03内(ボード画面)ではこれまでどおり画面遷移は発生せず、単一画面内のDOM操作で全ての機能を提供する

## 画面イメージ(ワイヤーフレーム)

簡易版のワイヤーフレームを以下に示す。実装時はこのレイアウトを基準とし、
配色・フォント等の詳細デザインは別途モックアップで確認する。

### S-01 ログイン画面 / S-02 新規登録画面

```
        +------------------------------+
        |          タスクボード         |
        +------------------------------+
        |  [ エラーメッセージ(任意) ]  |
        |  ユーザー名                   |
        |  [__________________________] |
        |  パスワード                   |
        |  [__________________________] |
        |  [        ログイン          ] |
        |  アカウントをお持ちでない方は  |
        |  新規登録                      |
        +------------------------------+
```

### S-03 ボード画面

```
+----------------------------------------------------------------------+
|  タスクボード  ○○さん          [ サンプルに戻す ] [ ログアウト ]     |
+----------------------------------------------------------------------+
|  To Do        ✕    |  進行中        ✕  |  完了          ✕  |           |
|  [優先度順][期限順] |  [優先度順][期限順]|  [優先度順][期限順]|  + リスト |
|  ------------      |  ------------     |  ------------     |    を     |
|  [カード1     ✕]  |  [カード3     ✕] |  [カード5     ✕] |    追加   |
|  [高 ▾][期限日]    |  [中 ▾][期限日]   |  [低 ▾][期限日]   |           |
|  [カード2     ✕]  |  [カード4     ✕] |                  |           |
|  [中 ▾][期限日]    |  [中 ▾][期限日]   |                  |           |
|  + カードを追加    |  + カードを追加   |  + カードを追加   |           |
+----------------------------------------------------------------------+
```

- カードはマウスドラッグで同一リスト内・他リストの任意の位置へ自由に並び替え・移動できる(F-07)
- リスト名・カード本文はクリックでその場編集ができる(F-02, F-05)
- カードの優先度・期限日は常時表示されるプルダウン/日付入力から直接変更できる(F-10, F-11)
- 各リストの「優先度順」「期限順」ボタンを押すと、そのリスト内のカードのみが一括で並び替わる(F-12)。押した瞬間の整列であり、以後は自由にドラッグして崩せる
- 一番右の「+ リストを追加」列は常に画面右端に固定表示する
- 未ログイン時はログイン画面(S-01)を表示し、ログイン成功後にボード画面(S-03)を表示する(F-13, F-14)
- ボード画面の「ログアウト」ボタンでログイン画面に戻る(F-15)
