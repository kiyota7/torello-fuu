# 機能一覧

タスク管理アプリ「タスクボード」の機能要件一覧。
全体の背景・目的は [要件定義書.md](../要件定義書.md) を参照。

## 機能要件

本システムが備える機能は以下の通りとする。

| 機能ID | 機能名 | 概要 |
|---|---|---|
| F-01 | リストの追加 | 「+ リストを追加」からリスト名を入力し、新しいリスト(列)を作成できる |
| F-02 | リスト名の編集 | リストのタイトル部分をクリックすると直接編集でき、変更内容が保存される |
| F-03 | リストの削除 | リスト右上の「✕」から、リストと配下の全カードを削除できる(確認ダイアログあり) |
| F-04 | カードの追加 | 各リスト下部の「+ カードを追加」からカード(タスク)を作成できる |
| F-05 | カード内容の編集 | カードのテキスト部分をクリックすると直接編集でき、変更内容が保存される |
| F-06 | カードの削除 | カードにカーソルを合わせると表示される「✕」から、確認なしで即時削除できる |
| F-07 | カードの移動(ドラッグ&ドロップ) | カードをドラッグし、任意のリストにドロップすることでカードの所属リストを変更できる |
| F-08 | データの自動保存 | リスト・カードの作成/編集/削除/移動内容は、ブラウザのlocalStorageに自動保存され、再訪問時も保持される |
| F-09 | 初期データへのリセット | 画面右上の「サンプルに戻す」から、保存データを削除しサンプルの初期状態に戻せる(確認ダイアログあり) |

## 機能対象外(スコープ外)とする項目

学校課題として簡易実装とするため、以下の機能は本システムの対象外とする。

- ユーザー登録・ログイン機能、複数ユーザーでの共同編集
- カードの期限日・担当者・ラベル・添付ファイルなどの詳細情報
- リストやカードの並び替え(同一リスト内での順序変更)
- サーバーサイドでのデータ保存、複数端末間でのデータ同期
- 検索・フィルタ・通知機能
