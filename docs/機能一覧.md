# 機能一覧

タスク管理アプリ「タスクボード」の機能要件一覧。
全体の背景・目的は [要件定義書.md](../要件定義書.md) を参照。

## 機能要件

本システムが備える機能は以下の通りとする。

| 機能ID | 機能名 | 概要 |
|---|---|---|
| F-01 | リストの追加 | 「+ リストを追加」からリスト名を入力し、新しいリスト(列)を作成できる |
| F-02 | リスト名の編集 | リストのタイトル部分をクリックすると直接編集でき、変更内容が保存される |
| F-03 | リストの削除 | リスト右上の「✕」から、リストと配下の全カードを削除できる(確認ダイアログあり) |
| F-04 | カードの追加 | 各リスト下部の「+ カードを追加」からカード(タスク)を作成できる |
| F-05 | カード内容の編集 | カードのテキスト部分をクリックすると直接編集でき、変更内容が保存される |
| F-06 | カードの削除 | カードにカーソルを合わせると表示される「✕」から、確認なしで即時削除できる |
| F-07 | カードの移動(ドラッグ&ドロップ) | カードをドラッグし、同一リスト内の任意の位置、または他のリストの任意の位置へ自由に並び替え・移動できる |
| F-08 | データの自動保存 | リスト・カードの作成/編集/削除/移動内容は、バックエンドAPIを通じてPostgreSQLデータベースに自動保存され、再訪問時も保持される |
| F-09 | 初期データへのリセット | 画面右上の「サンプルに戻す」から、保存データを削除しサンプルの初期状態に戻せる(確認ダイアログあり) |
| F-10 | カードの優先度設定 | カードに「高・中・低」の3段階の優先度を設定できる(新規作成時のデフォルトは「中」) |
| F-11 | カードの期限日設定 | カードに期限日を設定できる(未設定も可) |
| F-12 | 優先度順・期限順への並び替え | 各リストのヘッダーにある「優先度順」「期限順」ボタンを押すと、そのリスト内のカードが優先度(高→中→低)または期限日(昇順、未設定は最後)で一括して並び替えられる。並び替え後も自由なドラッグ操作(F-07)で順序を変更できる |

## 機能対象外(スコープ外)とする項目

学校課題として簡易実装とするため、以下の機能は本システムの対象外とする。

- ユーザー登録・ログイン機能、複数ユーザーによる同時編集時の排他制御(全利用者が同一のボードデータを共有する)
- カードの担当者・ラベル・添付ファイルなどの詳細情報(優先度・期限日はF-10, F-11として対象)
- リスト自体の並び替え(表示順の変更)
- 検索・フィルタ・通知機能
